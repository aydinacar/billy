# [BILLY](https://billy.vercel.app)

This is a simple invoicing app built with Next.js, TypeScript, Tailwind CSS, and Shadcn UI. It allows freelancers and small business owners to create and manage invoices, track payments, and visualize revenue with charts.

## Why this project?

Because I wanted to build a real-world application that incorporates a variety of technologies and features. I'm developing it for my portfolio and to use in my own freelance work. Existing solutions are either too expensive (FreshBooks, QuickBooks) or too simple (Excel). I wanted to create something that is affordable, user-friendly, and visually appealing.

## Tech Stack

[Next.js, TypeScript, Tailwind CSS, Shadcn UI, Neon, Clerk, Drizzle, PostgreSQL, Vercel, Stripe]

## Features (v1)

- [ ] Authentication
- [ ] Client management (CRUD)
- [ ] Invoice creation with multi-currency
- [ ] PDF export
- [ ] Payment tracking
- [ ] Revenue dashboard with charts
- [ ] Dark mode
